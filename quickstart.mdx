## In 3 Schritten zum intelligenten Telefonassistenten

Richten Sie HalloFelix ein und automatisieren Sie Ihre Telefonannahme in wenigen Minuten.

### Schritt 1: Neuen Assistenten erstellen

<Steps>
  <Step title="Anmelden und Dashboard öffnen">
    Melden Sie sich bei [HalloFelix](https://app.hallofelix.de) an und öffnen Sie Ihr Dashboard.
  </Step>
  <Step title="Neuen Assistenten anlegen">
    Klicken Sie auf **Neuer Assistent** und geben Sie einen Namen ein (z.B. "Empfang").
  </Step>
  <Step title="Grundeinstellungen konfigurieren">
    - Wählen Sie eine [Stimme](/funktionen/stimmen) aus (empfohlen: Viktoria oder Klaus)
    - Legen Sie den [ersten Satz](/funktionen/erster-satz) fest (z.B. "Guten Tag, Sie sprechen mit dem Büro von Max Mustermann. Wie kann ich Ihnen helfen?")
    - Stellen Sie die Gesprächsgeschwindigkeit ein (Normal empfohlen)
  </Step>
</Steps>

### Schritt 2: Rufweiterleitung einrichten

<AccordionGroup>
  <Accordion icon="mobile" title="Rufweiterleitung vom Handy">
    Leiten Sie Anrufe von Ihrem Handy zu Felix weiter:

    1. Kopieren Sie die Felix-Telefonnummer aus dem Dashboard
    2. Öffnen Sie die Telefon-App auf Ihrem Handy
    3. Geben Sie folgenden Code ein:
       ```
       **61*[Felix-Nummer]*15#
       ```
       (Ersetzt `[Felix-Nummer]` durch die tatsächliche Nummer)
    4. Drücken Sie die Anruftaste

    <Tip>
      Die Rufweiterleitung wird nach 15 Sekunden aktiv. Sie können die Zeit anpassen (5, 10, 15, 20, 25 oder 30 Sekunden).
    </Tip>

    Detaillierte Anleitung: [Rufweiterleitung am Handy](/rufweiterleitung/handy)
  </Accordion>
  <Accordion icon="phone-office" title="Rufweiterleitung über Telefonanlage">
    Wenn Sie eine Telefonanlage nutzen, finden Sie hier spezifische Anleitungen:

    - [Fritzbox](/rufweiterleitung/fritzbox)
    - [RingCentral](/rufweiterleitung/ringcentral)
    - [Aircall](/rufweiterleitung/aircall)
    - [Sipgate](/rufweiterleitung/sipgate)
    - [Placetel](/rufweiterleitung/placetel)
    - [Starface](/rufweiterleitung/starface)
  </Accordion>
</AccordionGroup>

### Schritt 3: Testen und optimieren

<Steps>
  <Step title="Testanruf durchführen">
    Klicken Sie im Dashboard auf **Assistent testen** und führen Sie einen Testanruf durch.
  </Step>
  <Step title="Öffnungszeiten konfigurieren">
    Legen Sie Ihre [Öffnungszeiten](/funktionen/oeffnungszeiten) fest, damit Felix außerhalb der Geschäftszeiten anders reagiert.
  </Step>
  <Step title="E-Mail-Benachrichtigungen einrichten">
    Konfigurieren Sie [E-Mail-Benachrichtigungen](/funktionen/email-benachrichtigung), um nach jedem Anruf eine Zusammenfassung zu erhalten.
  </Step>
  <Step title="Weiterleitungsnummer hinterlegen">
    Hinterlegen Sie eine [Weiterleitungsnummer](/funktionen/weiterleitungen), an die Felix bei Bedarf durchstellen kann.
  </Step>
</Steps>
